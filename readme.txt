== Installation ==

Installation using "Add New Plugin"
1. From your Admin UI (Dashboard), go to Plugins. Click the "Add New" button.
2. Search for plugin, or click the "Upload" button, and then select the plugin you want to install.
3. Click the "Install Now" button.

Manual installation
1. Upload the "without-update-themes" folder to the "/wp-content/plugins/" directory.

Settings
1. visit plugins => Without Update Themes Settings
2. check to without update theme
3. click save button
